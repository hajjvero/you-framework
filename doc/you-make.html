Documentation de la Bibliothèque YouMake

YouMake est un puissant générateur de code CLI pour YouFramework, permettant d'accélérer le développement en créant des structures de fichiers standardisées.

Commandes de Génération

Générer un Contrôleur

Utilisez make:controller pour créer un nouveau contrôleur et son template Twig associé.

php bin/console make:controller UserController
  • name : Nom du contrôleur (ex: UserController ou Admin/DashboardController).
  • --no-template : Option pour générer uniquement le contrôleur sans le fichier Twig.

Générer une Entité ORM

Utilisez make:entity pour créer une entité YouORM et son Repository.

php bin/console make:entity Product
  • name : Nom de l'entité.
  • --no-repository : Option pour ne pas générer de classe Repository.

Générer une Migration

Le système compare automatiquement vos entités PHP avec l'état actuel de votre base de données pour générer le code SQL nécessaire (up/down).

php bin/console make:migration

Générer une Commande Console

Créez vos propres outils CLI personnalisés héritant de YouConsole.

php bin/console make:command ExportDataCommand

Configuration des Chemins

YouMake respecte les chemins configurés dans vos fichiers de configuration (YouConfig) :

  • app.routes.resource : Destination des contrôleurs.
  • app.twig.path : Destination des templates Twig.
  • database.entities_path : Destination des entités.
  • database.repositories_path : Destination des repositories.
  • database.migrations_path : Destination des fichiers de migration.
